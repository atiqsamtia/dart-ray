# ray

Dart implementation of ray client for debugging dart apps faster on the fly.

## Installation
Add this package to your `pubspec.yaml` in `dependencies:` section
```yaml
dependencies: 
  ray: ^0.0.1
```
Update dependencies 
```
flutter pub get
```
